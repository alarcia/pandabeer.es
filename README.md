## Features

- Brand pages and beer catalog
- Events, news and promotions
- Lightweight ecommerce integration (Medusa.js backoffice)
- Tailwind-based UI and accessible markup

## Project layout

- `app/` — Next.js app routes and pages
- `public/` — static assets (images, fonts)
- `content/` — simple JSON content used by the site
